#  Clustering des Chiffres Manuscrits avec K-Means

##  Description du projet
Ce projet démontre l'utilisation de l'algorithme K-Means pour regrouper automatiquement des images de chiffres manuscrits sans connaître leurs étiquettes.

### Objectifs pédagogiques:
1. Comprendre le clustering non supervisé
2. Appliquer K-Means sur des données réelles
3. Résoudre le problème de correspondance des labels
4. Évaluer les performances du clustering
